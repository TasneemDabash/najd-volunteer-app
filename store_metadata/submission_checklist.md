# App Store Submission Checklist

## Apple App Store

### Before Submission
- [ ] App tested on multiple iOS devices
- [ ] No crashes or critical bugs
- [ ] All features working correctly
- [ ] Arabic and English languages tested
- [ ] Privacy Policy URL hosted and accessible
- [ ] Terms of Service URL hosted and accessible
- [ ] App icon (1024x1024) prepared
- [ ] Screenshots for all required device sizes
- [ ] App preview video (optional)

### App Store Connect Setup
- [ ] Bundle ID registered
- [ ] App name reserved
- [ ] Primary language set (Arabic)
- [ ] Category selected (Lifestyle)
- [ ] Age rating questionnaire completed
- [ ] App Privacy questionnaire completed
- [ ] Contact information added
- [ ] Support URL added
- [ ] Marketing URL added (optional)

### Required Screenshots
- [ ] iPhone 6.9" (iPhone 15 Pro Max) - Required
- [ ] iPhone 6.7" (iPhone 15 Plus) - Required
- [ ] iPhone 6.5" (iPhone 14 Plus) - Required
- [ ] iPhone 5.5" (iPhone 8 Plus) - Required
- [ ] iPad Pro 12.9" - If supporting iPad

### Metadata
- [ ] App name (30 characters max)
- [ ] Subtitle (30 characters max)
- [ ] Description (4000 characters max)
- [ ] Keywords (100 characters max)
- [ ] What's New text
- [ ] Promotional text (optional)

### Review Notes
- [ ] Demo account credentials (if login required)
- [ ] Instructions for testing
- [ ] Explanation of any special features

---

## Google Play Store

### Before Submission
- [ ] App tested on multiple Android devices
- [ ] No crashes or ANRs
- [ ] All features working correctly
- [ ] Arabic and English languages tested
- [ ] Privacy Policy URL hosted and accessible
- [ ] Signed APK/AAB generated

### Play Console Setup
- [ ] Developer account verified
- [ ] App created in Play Console
- [ ] App category selected
- [ ] Content rating questionnaire completed
- [ ] Data safety section completed
- [ ] Target audience set (13+)
- [ ] Contact details added

### Required Assets
- [ ] High-res icon (512x512)
- [ ] Feature graphic (1024x500)
- [ ] Screenshots (min 2, max 8) for phone
- [ ] Screenshots for tablet (if supporting)
- [ ] Promo video (optional)

### Store Listing
- [ ] App title (50 characters max)
- [ ] Short description (80 characters max)
- [ ] Full description (4000 characters max)
- [ ] App category
- [ ] Contact email
- [ ] Privacy policy URL

### Data Safety
- [ ] Data collection disclosed
- [ ] Data sharing disclosed
- [ ] Security practices confirmed
- [ ] Data deletion capability confirmed

### Testing
- [ ] Internal testing track tested
- [ ] Closed testing (optional)
- [ ] Open testing (optional)
- [ ] Pre-launch report reviewed

## Common Requirements

### Legal
- [x] Privacy Policy created (in-app)
- [x] Terms of Service created (in-app)
- [ ] Privacy Policy hosted online
- [ ] Terms of Service hosted online
- [x] GDPR compliance (data deletion capability)
- [x] Age restrictions disclosed

### Accessibility
- [ ] VoiceOver/TalkBack support tested
- [ ] Color contrast meets standards
- [ ] Touch targets appropriate size
- [ ] Text size supports dynamic type

### Performance
- [ ] App size optimized
- [ ] Battery usage optimized
- [ ] Network usage efficient
- [ ] Cold start time acceptable

### Security
- [x] HTTPS only (no HTTP)
- [x] Sensitive data encrypted
- [x] Authentication secure
- [ ] No sensitive data in logs
- [x] Permissions requested appropriately

## Post-Submission

### If Rejected
1. Read rejection reason carefully
2. Fix the issues mentioned
3. Respond to reviewer with clarification if needed
4. Resubmit with detailed release notes

### After Approval
- [ ] Verify app appears correctly in store
- [ ] Test download and installation
- [ ] Monitor crash reports
- [ ] Respond to user reviews
- [ ] Plan next update
