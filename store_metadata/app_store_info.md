## Description

### Arabic
نجد هو تطبيق شامل لإدارة وتنسيق العمل التطوعي. يربط المتطوعين بالمهام والمشاريع التي تحتاج مساعدتهم.

**المميزات الرئيسية:**
• إدارة المهام التطوعية وتتبع التقدم
• التواصل المباشر مع فريق الدعم والمنسقين
• تحديد المهارات والتوفر للمهام المناسبة
• مكالمات صوتية ومرئية مع فريق العمل
• تتبع ساعات التطوع والإنجازات
• إشعارات فورية للمهام الجديدة

**للمتطوعين:**
سجل كمتطوع، حدد مهاراتك، واستقبل المهام المناسبة لك.

**للمنسقين:**
أنشئ المهام، وزعها على المتطوعين، وتابع التقدم بسهولة.

انضم إلى مجتمع نجد وكن جزءاً من الفرق!

### English
Najd is a comprehensive app for managing and coordinating volunteer work. It connects volunteers with tasks and projects that need their help.

**Key Features:**
• Manage volunteer tasks and track progress
• Direct communication with support team and coordinators
• Specify skills and availability for suitable tasks
• Voice and video calls with team members
• Track volunteer hours and achievements
• Instant notifications for new tasks

**For Volunteers:**
Register as a volunteer, specify your skills, and receive tasks suited to you.

**For Coordinators:**
Create tasks, assign them to volunteers, and easily track progress.

Join the Najd community and make a difference!

# Google Play Data Safety

## Data Collection Summary

### Data Collected

| Data Type | Collected | Shared | Purpose |
|-----------|-----------|--------|---------|
| Name | Yes | No | Account management, Task coordination |
| Email | Yes | No | Account management, Communications |
| Phone Number | Optional | No | Emergency contact |
| Location | Yes | With coordinators | Match with nearby tasks |
| Photos/Videos | Yes | With coordinators | Task documentation |
| Voice recordings | Yes | With support team | Voice messages, Calls |
| App interactions | Yes | No | Analytics, Improvements |

### Data Security
- All data is encrypted in transit using TLS/SSL
- Data is stored on secure servers
- Users can request data deletion

### Data Handling
- **Retention**: Data retained while account is active
- **Deletion**: Users can delete their account and all associated data
- **Export**: Users can request a copy of their data
